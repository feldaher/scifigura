# Security Policy

## Supported Versions

Only the latest release (and current beta versions) of SciFigura are supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 0.1.0 | :white_check_mark: |
| < 0.1.0  | :x:                |

## Reporting a Vulnerability

We take the security of SciFigura seriously. If you discover a security vulnerability, please **do not open a public issue**. 

Instead, please report it via [GitHub's Private Vulnerability Reporting](https://github.com/feldaher/scifigura/security/advisories/new).

This allows us to review and address the vulnerability privately before it is publicly disclosed. We will do our best to acknowledge your report within 48 hours and keep you informed of our progress towards a fix.
